# devbox

Provision throwaway Ubuntu boxes as remote coding playfields.

## Layout

- [`ansible/`](ansible/) — the provisioning setup (roles, playbooks, scripts). See [ansible/README.md](ansible/README.md).
- [`cmd/hetzner/`](cmd/hetzner/) — small Hetzner Cloud API scripts to create/list/delete boxes.
- [`cmd/infomaniak/`](cmd/infomaniak/) — the same, for Infomaniak Public Cloud (OpenStack). See [cmd/infomaniak/README.md](cmd/infomaniak/README.md).
- [`AGENTS.md`](AGENTS.md) — notes for AI agents working in this repo.
